# Cataract Screening – Research Demo

This project demonstrates a calibrated ensemble deep learning system for
cataract screening and severity risk estimation.

### Model Outputs
- Cataract detected: Yes / No
- Severity risk (Immature vs Mature probabilities)
- Lens type (Natural / IOL)

### Notes
- Research demonstration only
- Not a medical device
- Designed for screening and referral support

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
